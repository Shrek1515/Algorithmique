----------------------------------------------------------
Pour lancer le programme avec des arbres binaires aléatoires :
1) compiler le code avec : gcc arbre_binaire.c parcours.c main.c -o aleatoire
2) lancer le programme avec : ./aleatoire NIV_NOEUDS avec NIV_NOEUDS la profondeur de l'arbre

EXEMPLE :

faire ./aleatoire 10 pour créer et afficher le parcours d'un arbre binaire aléatoire de profondeur 10
